# HaloSight Safety

HaloSight is external capture only.

It does not:

- inject into Halo
- read game memory
- bypass anti-cheat
- manipulate input
- close or clean browsers
- change Halo priority
- terminate game, browser, capture, overlay, or chat processes

It collects external Windows state, process/service snapshots, device status, recent system errors, and evidence files created by your own capture tools.
