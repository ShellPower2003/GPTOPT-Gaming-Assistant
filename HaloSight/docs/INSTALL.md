# Install HaloSight

## From the GitHub repo

1. Clone or download `GPTOPT-Gaming-Assistant`.
2. Open the repo folder on Windows.
3. Run `GPTOPT_LAUNCHER.cmd`.
4. The HaloSight GUI opens directly.

## From the standalone package

1. Extract the HaloSight zip.
2. Open the extracted folder.
3. Run `GPTOPT_LAUNCHER.cmd`.
4. The HaloSight GUI opens directly.

No installer is required. HaloSight writes session output under `%USERPROFILE%\Documents\GPTOPT\HaloSight\sessions` unless you change the setting in the GUI.
