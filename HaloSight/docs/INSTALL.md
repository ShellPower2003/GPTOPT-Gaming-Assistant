# Install HaloSight

## From the GitHub repo

1. Clone or download `GPTOPT-Gaming-Assistant`.
2. Open the repo folder on Windows.
3. Run `GPTOPT_LAUNCHER.cmd` from the repo root, or open the `HaloSight` folder and run `HaloSight_GUI.cmd`.

## From the standalone package

1. Extract the HaloSight zip.
2. Open the extracted folder.
3. Run `GPTOPT_LAUNCHER.cmd` or `HaloSight_GUI.cmd`.

No installer is required. HaloSight writes session output under `%USERPROFILE%\Documents\GPTOPT\HaloSight\sessions` unless you change the setting.
