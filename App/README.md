# GPTOPT App Launchers

This folder contains repo-owned launchers for GPTOPT.

Use `GPTOPT-Control.cmd` as the app-style entry point.

The launchers are intended to call the existing safe `Run-GPTOPT.ps1` modes from the repository root. They are preview-oriented and should not apply live fixes, edit Halo settings, run third-party tweak scripts, or reboot the machine.

Planned launchers:

- `GPTOPT-Control.cmd` - main menu / control center
- `GPTOPT-Audit.cmd` - read-only audit
- `GPTOPT-HaloSight.cmd` - HaloSight GUI
- `GPTOPT-Recommend.cmd` - NormalGaming recommendations
- `GPTOPT-PreviewQueue.cmd` - FullOptimizationPreview queue
- `GPTOPT-Report.cmd` - report bundle
- `GPTOPT-SafetyScan.cmd` - safety scanner
